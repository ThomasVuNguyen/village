# The experience of a user

First, install 'village' application
Sign up as user
Register a device
('village' automatically runs in the background)
Ask if user has another device.

If yes: 
  Turn on the other device
  Install 'village' application
  Sign in as user
  Register device #2
  ('village' automatically runs in the background)
  Send a command from device #1 to device #2
  
If no:
  User can send a command to one of the public devices
  Get a response
  
Get hooked
Share with friends
